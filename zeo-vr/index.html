Zeo VR

ZeoVR is a platform that lets you build immersive multiplayer WebVR experiences. It's built entirely out of JavaScript and plays well with NPM, with an open-source codebase for building plugins. The platform was built for developers and users alike — for users to have fun in VR with their friends without having to download an app, and for web developers to build plugins to make some extra cash on the side.

The ideal part about Zeo was the ability to run worlds on their own blockchain to build a better sharing economy.
